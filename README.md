# Hello World Turbinado

O objetivo é criar um programa simples que interaja com o usuário, pedindo **nome** e **idade**, e respondendo de forma personalizada.

---

## Como funciona

1. O programa pede ao usuário que digite seu **nome**.  
2. Em seguida, solicita a **idade**.  
3. Calcula aproximadamente quantos dias a pessoa já viveu (`idade * 365`).  
4. Exibe uma mensagem personalizada dependendo se o usuário é menor ou maior de idade.
